mft2014
=======

MakerFaireTokyo 2014 / 自転車で行くデスクトップの旅2

00_Sensor
- 自転車からのセンサー情報、コンパス情報を取得するMaxパッチ
- 下位のアプリにOSCメッセージを送信する

10_Controller
- 自転車情報をデスクトップ空間にマップする
- 下位のアプリに座標、方角を一定間隔ごとに送信する
- キーボードによるシミュレータも同梱

20_SoundPlayer
- デスクトップに配置されたサウンドファイルをミックスして再生
- ポインタからファイルへの距離を考慮

30_3dView

40_2dView

90_Batch
- デスクトップに配置されたファイルのFinder座標を取得します。
